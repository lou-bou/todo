## Repo Structure

##### index.js

Entry point for js files.

##### main.css

Entry point for css files.

##### logic.js

Classes and methods.

##### dom.js

All DOM stuff that gets executed.

##### formsHandling.js

Handles reading and clearing forms' data.

##### taskDOM.js

Handles tasks' DOM creation, editing, and rendering.

##### /css

All css sub-files.

### Naming Convention

If a variable has 'Object' in it, it's related to the task Object that is/will be/was stored in localStorage.
If a variable has 'DOM' in it, or no label, it's related to the task DOM element.
